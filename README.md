# JSBot
JavaScriptを推進するDiscordボット

# How to use
1, Append
```ts
import * as dotenv from "dotenv";
dotenv.config();
```
into "/src/server.ts"  

2, create ".env"  
```
DISCORD_TOKEN=(your token)
```
3, run  
```
npm install
npm run build
npm start
```
